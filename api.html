API Reference

This section documents the darca-llm API.

—

AIClient

darca_llm.AIClient(backend: str = 'openai')[source]

A unified client for interacting with LLMs using the darca pluggable backend system. Defaults to OpenAI.

Unified interface to interact with LLMs.

Parameters:

backend – The backend to use (default: “openai”)

Raises:

LLMException – If the backend is unsupported

—

OpenAIClient

darca_llm.OpenAIClient()[source]

LLM backend that uses OpenAI’s GPT models via their official API.

Handles direct interaction with OpenAI’s API. Inherits from BaseLLMClient.

—

BaseLLMClient

darca_llm.BaseLLMClient()[source]

Abstract base class for LLM backends. Provides shared logic for file content processing.

Abstract base class for all LLM clients.

Methods:

  • get_raw_response(system, user, …)

  • get_file_content_response(system, file_content, …)

darca_llm.LLMAPIKeyMissing(message, error_code=None, metadata=None, cause=None)[source]

darca_llm.LLMContentFormatError(message, error_code=None, metadata=None, cause=None)[source]

darca_llm.LLMResponseError(message, error_code=None, metadata=None, cause=None)[source]
